## How to Export Selected Features in QGIS

One helpful technique in GIS software is the ability to quickly create filtered subsets of an original data source. You might want a new datasets that only has features of a certain type (e.g. only water bodies of a certain depth), *or* you may want to create a new dataset based on a subset of the geography.

In this tutorial, we will do the latter. We will start with a dataset of towns in Massachusetts, and export a new dataset that only shows the boundary of Cambridge, MA.
