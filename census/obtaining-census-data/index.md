# How to Obtain Census Data for GIS Mapping

Sometimes new mappers are tripped up by census data, because in order to make a map, you  need to download two separate census files:
1. A table of statistics
2. A geography boundary file

## Sources

In this lesson, you will learn how to get those files from two different sources, **Social Explorer** and **NHGIS.org**.

Social Explorer, while slightly more user-friendly, requires a license. You are able to [access Social Explorer with your Harvard Key](http://nrs.harvard.edu/urn-3:hul.eresource:socialex). 

If you do not have a Social Explorer license, the same data can be accessed on [NHGIS.org](nhgis.org).

Both of these options as sources yield results for census data that are better-suited for GIS methods than data that comes from the official census download website.

You will learn how to use both in this tutorial.

## Example data
For this tutorial series, we will be working with `tenure` data, which tells us for each census geography the percent renter vs. owner occupied.
